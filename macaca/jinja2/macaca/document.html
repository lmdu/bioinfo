(2) Quick search

Quick search function allows users to provide a SNP identifier (individual, group-specific, species-specific or non-redundant SNP), macaque Ensembl gene ID, OMIM phenotype accession number or drug ID from Drugbank database to query item.

Individual SNPs

The individual SNPs browse pape includs the filtration pane (1) with the SNP result table (2) located under it. Through filtration pane, users may filter snps in table by specify chrosome number, individual, gene feature (CDS, UTR or Intron etc.), genotype (homozygote or heterozygote) and mutation type (non-synonymous or synonymous). Click SNP ID hyperlink, users can obtain detailed information for SNP.
